# 已知问题与修复记录

## Issue #1: 游戏中无法查看自己身份
- **状态**: 已修复
- **描述**: 游戏进行中玩家没有醒目的地方查看自己的身份角色
- **原因**: PhaseDisplay右上角有小字显示，但不够醒目，容易忽略
- **修复**: 在游戏界面添加始终可见的"我的身份"卡片

## Issue #2: 夜晚行动回合太快
- **状态**: 已修复
- **描述**: AI给出选择太快(1-3秒)，真人玩家容易推测出谁是AI
- **原因**: aiDelayMs默认1-3秒，AI行动后立即推进到下一阶段
- **修复**: 夜间阶段引入最低15秒持续时间，白天发言最长3分钟
- **超时默认操作**: 预言家不验人、女巫不用药、狼人空刀、守卫不守护

## Issue #3: 白天讨论阶段游戏卡死 (重大Bug)
- **状态**: 已修复
- **描述**: 进入白天第一个回合后游戏卡死，玩家无法操作
- **原因**:
  - 讨论阶段使用发言队列(speakerQueue)，需要currentSpeaker完成后推进
  - 当人类玩家是currentSpeaker时，前端没有"结束发言"按钮
  - 服务端已实现advance_speaker事件，但客户端无UI触发
  - 只能等待120秒超时自动推进，用户体验极差
- **修复**: 添加"结束发言"按钮，玩家可主动推进发言轮次
